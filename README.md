# Voxon AI Chatbot

This project is a simple web based chatbot interface that connects to an AI model through a backend server. The frontend is built using HTML CSS and JavaScript. The backend handles all API calls securely so no keys are exposed in the client side code.

## Features
Chat interface with clean UI  
Backend request handling  
Secure environment variable setup for API keys  
Easy to extend and add new features

## How to run
Clone the repository  
Install backend dependencies  
Add your API key in the server environment file  
Start the server and open the frontend page in your browser

## Note
The API key is stored in the backend only. No keys are included in this repository.
